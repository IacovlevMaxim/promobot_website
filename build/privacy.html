What data does Promobot store?

Promobot Privacy Policy

Server info:

If you didnt setup the server, Promobot doesnt store any data about this server and only has access to data that is available through Discord API.


If you completed the setup, Promobot stores:

  • Server ID (accessable through Discord API)
  • Promo channel ID (provided during setup, accessable through Discord API)
  • Promo message (provided during setup)
  • Category of this server (provided during setup)
  • Discord invite to that server (provided during setup)
  • How many times this server was promoted in last 24 hours, counting from 00:00 UTC (updated everytime pr$promo command was successfully completed)
  • Timestamp of last promotion (updated everytime pr$promo command was successfully completed)
  • Is this server banned from using Promobot or not

If your server applied for whitelisting and got successfully accepted for it, Promobot additionally stores:

  • Promo channel ID that should be whitelisted
  • Server ID that should be whitelisted
  • Role ID(-s) that should be whitelisted

If you would like to get this data removed from the database, remove Promobot from your Discord server and all data will be cleared. However, if your server was banned from Promobot usage, ban state data will not be removed and will be stored forever to prevent using the bot in the future.


User info:

If a user never voted for Promobot on top.gg and never submitted Promobot whitelist request, Promobot doesnt store any data about this user and only access data that is available through Discord API.


If you voted for Promobot on top.gg, Promobot stores:

  • ID of user who voted (received from Top.gg API)
  • Timestamp of last vote (calculated when vote is received by the bot)
  • Whether user used the vote or not (updated everytime user uses pr$promo command)

If you would like to get this data removed from the database, please join Promobot support server and DM server owner with your request.


What data is publically available?

Everything that can be accessed through Discord app or through Discord API is counted as publically available. Any info that is not publically available is not revealed to any other individuals or third-party services, except for any statements in Privacy Policy that make exceptions for it.


Promobot may display links to certain third party websites, such as top.gg. I encourage you to read the individual privacy policies of those third party websites before providing any of your personal information to them.


I may change privacy policy from time to time. Latest version of Promobot Privacy Policy will be posted here. I encourage you to review our Privacy Policy on a regular basis so that you will be aware of any changes to it.
