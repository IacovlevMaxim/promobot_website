Promobot Commands

Setup commands

All setup commands are only usable in a guild

Command Aliases Description Example User Permissions Bot Permissions
setup start Sets up the bot on current server (bot will walk you through all necessary steps)
Note: click here to see setup tutorial 		pr$setup Manage Channels, Manage Messages View Channel, Send Messages, Embed Links, Create Instant Invite, Manage Server
category setCategory Sets up category of current server pr$category rocketleague View Channel, Send Messages, Embed Links
channel setChannel Channel where messages from other servers will be sent. pr$channel

#promo
invite setInvite Sets the invite that will be promoted together with the promotion message pr$invite https://discord.gg/dXES6RYtAq
message setMessage Sets the self promo message of this server. pr$message Your promo message!

Info commands

Command Aliases Description Example Required Permissions
categories sections Prints all currently valid categories pr$categories View Channel, Send Messages
addme None Sends an invite link to add the bot pr$addme
info Embed that shows bot's commands and basic usage pr$info View Channel, Send Messages, Embed Links
myVote Sends info about user's last vote pr$myVote
permissions Why does PromoBot require these permissions? pr$permissions
vote Sends an invite link to vote for the bot on top.gg pr$vote
privacy privacyPolicy, policy Sends Promobot Privacy Policy pr$privacy

Utility commands

Command Description Example Required Permissions
ping Pong! Checks the bot's ping to the Discord server. pr$ping View Channel, Send Messages
prefix Shows or sets the command prefix. @PromoBot#7792 prefix
pr$prefix
help Displays a list of available commands, or detailed information for a specified command. pr$help
pr$help ping
PromoBot

PromoBot is discord bot oriented to help smaller communities grow. It splits all servers in categories, and promotes your server where users will find it most relevant!
